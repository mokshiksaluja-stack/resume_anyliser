AI Resume Analyzer Project

This application was developed as an industry-standard, fully functional final year college major-project. It showcases deep full-stack engineering skills, combining real-time file stream parsing, dynamic regular-expression scanners, and customized dictionary-based Natural Language Processing.

Algorithmic Scoring Formula

The core score indexer runs an advanced multi-weighted scoring mechanism:
ATS Score = (KeywordOverlaps × 0.35) + (Skills Breadth × 0.25) + (StructuralMetrics × 0.20) + (ExperienceDepth × 0.20)

Unlike basic parsers that look for exact formatting strings, our vocabulary engine scans context words case-insensitively using anchor barriers, isolating technology groups dynamically into languages, cloud, frontend, and backend tiers.

Major Project Objectives

  • Demonstrate end-to-end full stack JWT architecture.
  • Implement local file JSON document fallbacks to guarantee 100% database recovery offline.
  • Incorporate client-side Single Page Application (SPA) state-routers.
  • Integrate micro-interactive analytics dashboards (Chart.js / gauges).

System Architecture Map
